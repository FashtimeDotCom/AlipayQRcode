# AlipayQRcode
个人收款码支付方案


预先自动生成个人收款二位码 相同金额每种生产一定的数量 根据交易量来决定
备注为： 商户号-商户支付宝编号-金额-金额编号
把图片扫描为地址，入库

本地： 监听支付宝到账通知 获取订单信息 发送给服务端
服务器： 接受支付申请，生成内部订单，返回支付地址，接收本地通知，返回支付状态给商户
商户：发起订单，接受回调，查询订单
